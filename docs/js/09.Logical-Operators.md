# 逻辑运算符 

布尔运算符

## 与（并且）

符号：`&&`

> 写法：表达式1 && 表达式2

1. 将表达式1 进行 boolean 判定

以下数据均判定为false：

- null
- undefined
- false
- NaN
- ''
- 0

其他数据全部为真

2. 如果表达式1 的判定结果为假，则直接返回表达式1，而不执行表达式2；否则，返回表达式2 的结果。（短路规则）

## 或

符号：`||`

> 写法：表达式1 || 表达式2

1. 将表达式1 进行 boolean 判定

2. 如果表达式为真，直接返回表达式1，不运行表达式2；否则，返回表达式2

## 非

符号：`!`

> 写法：!数据

一元运算符

将数据的boolean判定结果直接取反，非运算符一定返回boolean类型的值。

## 三目运算符

> 写法：表达式1 ? 表达式2 : 表达式3

1. 对表达式1 进行 boolean 判定
2. 如果判定结果为真，返回表达式2；否则，返回表达式3。
